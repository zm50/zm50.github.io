# 题解思路
定义递归深度deep，数字使用情况used，选择的数字顺序path

进行递归

终止条件为递归深度达到n层时，打印path，然后返回

深度加一
遍历未使用的数字，选择数字，然后进行递归，递归结束，恢复used
恢复深度

直到整个递归结束，程序结束
# 题解代码
```python
n = int(input())

used = 0
deep = 0
path = [0 for _ in range(n)]
def dfs():
    global used
    global deep
    if deep == n:
        for i in range(n):
            print(path[i], end=' ')
        print()
        return
    deep += 1
    for i in range(n):
        if used >> i & 1 == 0:
            path[deep - 1] = i + 1
            used |= 1 << i
            dfs()
            used -= 1 << i
    deep -= 1
dfs()
```
